# BugiChat

## 프로젝트 소개
ObjectStream을 사용한 자바 GUI 채팅 프로그램

## 프로젝트 기능
1. 로그인
2. 친구 추가 기능
3. 프로필 사진 변경
4. 개인, 단체 채팅방 만들기
5. 채팅 보내기
6. 이모티콘, 이미지 보내기
7. 프로필 사진, 채팅 이미지 원본 사진 보기
8. 메시지 저장
